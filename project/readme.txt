This project is an sample of a basic hotel reservation tool.

The project starts from the index page

Where you can either regsiter a new account or login from an ezisting account

After that you are redirected to about page from where you can go to the reservation page

In the reservation page you can input the details and the data is taken from the user and stored in the database

The staff can check the reservations from the list page

The database'hotel' consists of two tables

Registration and Reservation

The registered account data is stored in the registration database and the login compare the data from the same table

The reservation table has the data of reservations and displays the same data in the list page.
